# Personal CV — LaTeX source

This repository contains the LaTeX source for my personal Curriculum Vitae (CV) and a cover letter template.

## Overview

- Main CV source: `cv-on.tex`
- Styling and macros: `settings.sty`
- Sections: `education.tex`, `employment.tex`, `projects.tex`, `skills.tex`
- Bibliography (optional): `own-bib.bib`
- Cover letter and assets: `cover_letter/`
